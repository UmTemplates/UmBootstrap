description: Full-width single-column layout.
---

A full-width layout with a single area spanning all 12 columns.

## Details

| Property | Value |
|---|---|
| Element type alias | `layout12` |
| Display name | Layout - 12 |
| Areas | 1 |
| Column split | 12 |

## Areas

| Alias | Description |
|---|---|
| (default) | Full-width area — all features allowed |

## Usage

Use `layout12` when content should span the full width of the page. This is the most common layout for simple content sections — a rich text block, an image, or a contact form.
