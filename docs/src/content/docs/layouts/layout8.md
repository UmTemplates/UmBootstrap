title: Layout 8
description: Narrower centred single-column layout.
---

A narrower centred layout with a single area spanning 8 of 12 columns.

## Details

| Property | Value |
|---|---|
| Element type alias | `layout8` |
| Display name | Layout - 8 |
| Areas | 1 |
| Column split | 8 (centred) |

## Areas

| Alias | Description |
|---|---|
| (default) | Centred area — narrower than full width |

## Usage

Use `layout8` when you want content to be narrower than full width for better readability. Good for long-form text content where full-width lines would be too wide, or for forms that don't need the full page width.
