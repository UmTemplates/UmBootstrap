An equal two-column layout splitting the page 50/50.

## Details

| Property | Value |
|---|---|
| Element type alias | `layout66` |
| Display name | Layout - 6 \| 6 |
| Areas | 2 |
| Column split | 6 + 6 |

## Areas

| Alias | Desktop columns | Purpose |
|---|---|---|
| (left) | 6 | Left content area |
| (right) | 6 | Right content area |

## Responsive Behaviour

- **Desktop (md+)**: Two equal columns side by side
- **Mobile (below md)**: Both areas stack to full width

## Usage

Use `layout66` for side-by-side content of equal importance — for example, an image next to text, or two feature blocks that complement each other.
