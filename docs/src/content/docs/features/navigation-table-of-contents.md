---
title: Navigation - Table of Contents
description: Displays a table of contents navigation for the current page.
---

The Navigation - Table of Contents feature renders a table of contents for the current page.

## Details

| Property | Value |
|---|---|
| Element type alias | `featureNavigationTableOfContents` |
| Display name | Navigation - Table of Contents |
| Component composition | `featureComponentNoConfiguration` |
| Settings type | `featureSettings` |
| Uses shared layout | Yes (`_Layout_Features.cshtml`) |

## How It Works

This is a **no-configuration feature** — it composes only `featureComponentNoConfiguration` and derives its content from the page context.
