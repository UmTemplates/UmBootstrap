The Image Slideshow feature displays a Bootstrap carousel populated with images from the media library.

## Details

| Property | Value |
|---|---|
| Element type alias | `featureImageSlideshow` |
| Display name | Image Slideshow |
| Component composition | `featureComponentImageSlideshow` |
| Settings type | `featureSettings` |
| Uses shared layout | Yes (`_Layout_Features.cshtml`) |

## Compositions

- `featureComponentFeatureTitle` — heading
- `featureComponentFeatureDescription` — lead text
- `featureComponentFeatureSummary` — footer text
- `featureComponentImageSlideshow` — media picker (multiple images)
