---
title: Internal Links - Pagination
description: Display internal page links with pagination support.
---

The Internal Links - Pagination feature displays internal page links with pagination controls for longer lists.

## Details

| Property | Value |
|---|---|
| Element type alias | `featureInternalLinksPagination` |
| Display name | Internal Links - Pagination |
| Component composition | `featureComponentsInternalLinks` |
| Settings type | `featureSettings` |
| Uses shared layout | Yes (`_Layout_Features.cshtml`) |

## Compositions

- `featureComponentFeatureTitle` — heading
- `featureComponentFeatureDescription` — lead text
- `featureComponentFeatureSummary` — footer text
- `featureComponentsInternalLinks` — shared internal links composition
