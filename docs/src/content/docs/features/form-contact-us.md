The Form - Contact Us feature provides a simple contact form with inline form fields.

## Details

| Property | Value |
|---|---|
| Element type alias | `featureFormContactUs` |
| Display name | Form - Contact Us |
| Component composition | (inline form fields) |
| Settings type | `featureSettings` |
| Uses shared layout | Yes (`_Layout_Features.cshtml`) |

## Compositions

- `featureComponentFeatureTitle` — heading
- `featureComponentFeatureDescription` — lead text
- `featureComponentFeatureSummary` — footer text

Unlike other features, the form fields are defined directly on the element type rather than through a separate component composition.
