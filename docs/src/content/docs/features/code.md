---
title: Code
description: Code snippet block for displaying formatted source code.
---

The Code feature lets editors add formatted code snippets to a page.

## Details

| Property | Value |
|---|---|
| Element type alias | `featureCode` |
| Display name | Code |
| Component composition | `featureComponentCode` |
| Settings type | `featureSettings` |
| Uses shared layout | Yes (`_Layout_Features.cshtml`) |

## Compositions

- `featureComponentFeatureTitle` — heading
- `featureComponentFeatureDescription` — lead text
- `featureComponentFeatureSummary` — footer text
- `featureComponentCode` — code content
