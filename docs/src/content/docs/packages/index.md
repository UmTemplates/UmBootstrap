UmBootstrap includes the following Umbraco community packages out of the box.

| Package | Version | Description |
|---------|---------|-------------|
| [BlockPreview](/UmBootstrap/packages/block-preview/) | 5.1.0 | Live block previews in the backoffice |
| [UmbNav](/UmBootstrap/packages/umbnav/) | 4.0.3-beta0001 | Navigation menu builder |
| [uSync](/UmBootstrap/packages/usync/) | 17.0.2 | Content and configuration synchronisation |
