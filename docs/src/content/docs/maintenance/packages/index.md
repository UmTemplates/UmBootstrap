title: Installed Packages
---

UmBootstrap includes the following Umbraco community packages out of the box.

| Package | Version | Description |
|---------|---------|-------------|
| [BlockPreview](/UmBootstrap/maintenance/packages/block-preview/) | 5.3.2 | Live block previews in the backoffice |
| [Contentment](/UmBootstrap/maintenance/packages/contentment/) | 6.1.1 | Custom data sources and property editors |
| [UmbNav](/UmBootstrap/maintenance/packages/umbnav/) | 4.1.5 | Navigation menu builder |
| [uSync](/UmBootstrap/maintenance/packages/usync/) | 17.0.4 | Content and configuration synchronisation |
