# Packages

UmBootstrap includes the following Umbraco community packages out of the box.

| Package | Version | Description |
|---------|---------|-------------|
| [BlockPreview](block-preview.md) | 5.1.0 | Live block previews in the backoffice |
| [UmbNav](umbnav.md) | 4.0.3-beta0001 | Navigation menu builder |
| [uSync](usync.md) | 17.0.2 | Content and configuration synchronisation |
